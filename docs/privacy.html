Privacy Policy

ChatGPT Prompt Manager Chrome Extension

Last Updated: November 23, 2024

Summary: This extension does not collect any personal data. All your prompts and settings are stored locally on your device.

1. Information We Do NOT Collect

ChatGPT Prompt Manager does not collect, transmit, or share any user data, including but not limited to:

  • Personal information (name, email, address, etc.)
  • Browsing history
  • User activity or behavior
  • Location data
  • Device information
  • Analytics or usage statistics

2. Local Data Storage

All data created by the extension is stored locally on your device using Chrome's local storage API (chrome.storage.local):

  • Prompt Templates: Your custom prompt templates and content
  • Settings: Your preferences (language, categories, etc.)
  • Usage Statistics: Local count of how many times you've used each prompt (stored only on your device)

Important: This data never leaves your device and is not transmitted to any external servers.

3. Permissions Explanation

The extension requires the following permissions:

  • storage: To save your prompts and settings locally on your device
  • activeTab: To insert prompts into the ChatGPT input field when you click on them
  • host_permissions (chat.openai.com, chatgpt.com): To detect ChatGPT pages and enable the floating prompt button

These permissions are used solely for the extension's functionality and not for data collection.

4. Third-Party Services

This extension does not use any third-party services, including:

  • No analytics tools (Google Analytics, etc.)
  • No tracking or telemetry
  • No external APIs or servers
  • No advertisements

5. Data Security

Since all data is stored locally on your device:

  • Your data is as secure as your device
  • No data is transmitted over the network
  • No risk of server breaches or unauthorized access
  • The extension follows Chrome's security best practices

6. Your Control

You have complete control over your data:

  • Export: Export all your prompts as JSON files at any time
  • Import: Import prompts from JSON files
  • Delete: Delete individual prompts or all data
  • Uninstall: Uninstalling the extension will remove all locally stored data

7. Open Source

This extension is open source. You can review the complete source code to verify our privacy practices at:

https://github.com/GsirGinRay/chatgpt-prompt-manager

8. Changes to This Policy

If we make any changes to this privacy policy, we will update the "Last Updated" date at the top of this page. We will also notify users through the extension if there are significant changes.

9. Contact

If you have any questions about this privacy policy, please contact us through:

  • GitHub Issues: Report an Issue

Bottom Line: We respect your privacy. This extension is designed to work entirely on your device without collecting or transmitting any data.
