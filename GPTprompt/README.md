# ChatGPT 提示詞管理工具

一個功能強大的 Chrome 擴充功能，幫助您管理和快速使用 ChatGPT 提示詞模板，支援自訂變數和快速插入功能。

## ✨ 主要功能

### 📝 提示詞管理
- 新增、編輯、刪除提示詞模板
- 支援分類和標籤管理
- 快速搜尋功能
- 使用次數統計

### 🔄 變數替換
- 使用 `[變數名稱]` 格式標記可替換內容
- 使用時自動彈出變數輸入框
- 支援多個變數的提示詞模板

### 💾 匯入/匯出
- 匯出所有提示詞為 JSON 檔案
- 支援匯入外部提示詞
- 可選擇合併或覆蓋現有資料

### ⚡ 快速插入
- 一鍵插入提示詞到 ChatGPT
- 支援 chat.openai.com 和 chatgpt.com
- 頁面浮動快速訪問按鈕

## 📦 安裝方式

### 方式一：從原始碼安裝（推薦）

1. **下載專案檔案**
   - 下載或複製整個專案資料夾

2. **準備圖示檔案**
   - 在 `icons` 資料夾中準備以下尺寸的圖示：
     - `icon16.png` (16x16 像素)
     - `icon48.png` (48x48 像素)
     - `icon128.png` (128x128 像素)
   - 或使用線上工具生成圖示

3. **載入擴充功能**
   - 開啟 Chrome 瀏覽器
   - 前往 `chrome://extensions/`
   - 開啟右上角的「開發人員模式」
   - 點擊「載入未封裝項目」
   - 選擇專案資料夾

4. **開始使用**
   - 前往 [ChatGPT](https://chat.openai.com) 或 [ChatGPT](https://chatgpt.com)
   - 點擊瀏覽器工具列的擴充功能圖示
   - 開始管理您的提示詞！

## 🚀 使用教學

### 1️⃣ 新增提示詞

1. 點擊擴充功能圖示開啟管理介面
2. 點擊「+ 新增提示詞」按鈕
3. 填寫提示詞資訊：
   - **名稱**：為提示詞取個好記的名字
   - **分類**：選填，用於組織提示詞
   - **內容**：輸入提示詞模板

### 2️⃣ 使用變數

在提示詞內容中使用 `[變數名稱]` 格式標記可替換的部分：

```
請幫我撰寫一篇關於 [主題] 的文章，
目標受眾是 [受眾]，
文章長度約 [字數] 字。
```

使用時會自動彈出輸入框讓您填寫變數值。

### 3️⃣ 使用提示詞

**方式一：從管理介面**
1. 開啟提示詞管理器
2. 點擊想要使用的提示詞卡片
3. 如果有變數，填寫變數值
4. 自動插入到 ChatGPT 輸入框

**方式二：快速訪問按鈕**
1. 在 ChatGPT 頁面右下角有浮動按鈕
2. 點擊開啟提示詞列表
3. 選擇並使用提示詞

### 4️⃣ 搜尋提示詞

在搜尋框輸入關鍵字，可以搜尋：
- 提示詞名稱
- 提示詞內容
- 分類標籤

### 5️⃣ 匯出與匯入

**匯出：**
1. 點擊右上角的下載圖示
2. 選擇儲存位置
3. 檔案會以 `chatgpt-prompts-YYYY-MM-DD.json` 格式儲存

**匯入：**
1. 點擊右上角的上傳圖示
2. 選擇之前匯出的 JSON 檔案
3. 提示詞會自動合併到現有列表

## 📋 提示詞範例

### 範例 1：SEO 文章撰寫
```
請幫我撰寫一篇關於 [主題] 的 SEO 優化文章，
目標關鍵字是 [關鍵字]，
目標受眾是 [受眾]，
文章長度約 [字數] 字，
語氣要 [語氣風格]。

請確保文章包含：
1. 吸引人的標題
2. 清晰的段落結構
3. 適當的關鍵字密度
4. 實用的內容價值
```

### 範例 2：程式碼審查
```
請幫我審查以下 [程式語言] 程式碼，重點關注：

1. 程式碼品質和可讀性
2. 潛在的 bug 或錯誤
3. 效能優化建議
4. 安全性問題
5. 最佳實踐建議

程式碼：
[程式碼內容]
```

### 範例 3：社群媒體貼文
```
請為 [平台] 撰寫一則關於 [主題] 的社群媒體貼文。

目標受眾：[受眾]
貼文語氣：[語氣]
字數限制：[字數]

請包含：
- 吸引人的開頭
- 清晰的價值主張
- 適當的標籤建議
- 行動呼籲（CTA）
```

## 🛠️ 技術架構

### 專案結構
```
提示詞外掛-chrome extention-prompt plugin/
├── manifest.json          # 擴充功能設定檔
├── popup.html            # 彈出視窗 HTML
├── popup.css             # 彈出視窗樣式
├── popup.js              # 彈出視窗邏輯
├── content.js            # 注入到 ChatGPT 頁面的腳本
├── content.css           # 注入到 ChatGPT 頁面的樣式
├── background.js         # 背景服務
├── storage.js            # 資料儲存管理模組
├── icons/                # 圖示資料夾
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
└── README.md             # 說明文件
```

### 技術特點
- **Manifest V3**：使用最新的擴充功能 API
- **Chrome Storage API**：本地資料持久化
- **模組化設計**：清晰的程式碼結構
- **響應式介面**：適配不同螢幕尺寸
- **無第三方依賴**：純原生 JavaScript 實作

## 🔧 開發說明

### 修改和自訂

1. **修改樣式**
   - 編輯 `popup.css` 改變管理介面外觀
   - 編輯 `content.css` 改變頁面按鈕樣式

2. **新增功能**
   - `storage.js`：資料管理相關功能
   - `popup.js`：介面互動邏輯
   - `content.js`：頁面注入功能

3. **調整 ChatGPT 選擇器**
   - 如果 ChatGPT 頁面結構改變
   - 修改 `content.js` 中的選擇器陣列

### 除錯方式

1. **檢查背景頁面**
   - 前往 `chrome://extensions/`
   - 點擊「服務工作程式」查看日誌

2. **檢查彈出視窗**
   - 在彈出視窗上右鍵選擇「檢查」
   - 查看 Console 輸出

3. **檢查 Content Script**
   - 在 ChatGPT 頁面按 F12
   - 查看 Console 輸出

## 📝 常見問題

### Q: 無法插入提示詞到 ChatGPT？
**A:** 請確認：
1. 您在 ChatGPT 對話頁面（不是首頁）
2. 頁面已完全載入
3. 重新整理頁面後再試

### Q: 提示詞資料會遺失嗎？
**A:** 不會，所有資料都儲存在本地，建議定期匯出備份。

### Q: 可以在其他網站使用嗎？
**A:** 目前僅支援 ChatGPT，如需支援其他 AI 平台，可修改 `manifest.json` 中的 `host_permissions`。

### Q: 如何建立圖示？
**A:** 可以使用以下方式：
1. 使用設計工具（如 Figma、Canva）建立
2. 使用線上圖示產生器
3. 使用 AI 繪圖工具生成
