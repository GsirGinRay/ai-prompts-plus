# 🚀 快速開始指南

## 5 分鐘快速部署

### 第 1 步：準備圖示（2 分鐘）

**最快方式 - 使用線上工具：**

1. 前往：https://favicon.io/emoji-favicons/speech-balloon/
2. 點擊「Download」下載圖示
3. 解壓縮 ZIP 檔案
4. 將以下檔案複製到 `icons` 資料夾：
   - `favicon-16x16.png` → 重新命名為 `icon16.png`
   - `android-chrome-192x192.png` → 調整為 48x48 並命名為 `icon48.png`
   - `android-chrome-512x512.png` → 調整為 128x128 並命名為 `icon128.png`

**或使用簡單替代方案：**

建立 3 個純色 PNG 檔案（使用任何線上圖片編輯器）：
- 尺寸：16x16、48x48、128x128
- 背景色：#10a37f
- 儲存為：icon16.png、icon48.png、icon128.png

### 第 2 步：檢查檔案（1 分鐘）

確認專案資料夾包含以下檔案：

```
✅ manifest.json
✅ popup.html
✅ popup.css
✅ popup.js
✅ storage.js
✅ content.js
✅ content.css
✅ background.js
✅ icons/icon16.png
✅ icons/icon48.png
✅ icons/icon128.png
```

### 第 3 步：安裝擴充功能（2 分鐘）

1. 開啟 Chrome，前往：`chrome://extensions/`
2. 開啟右上角「開發人員模式」
3. 點擊「載入未封裝項目」
4. 選擇專案資料夾
5. 完成！

### 第 4 步：測試（1 分鐘）

1. 點擊擴充功能圖示
2. 應該看到 3 個預設範例提示詞
3. 前往 https://chat.openai.com
4. 右下角應該有綠色浮動按鈕
5. 測試插入一個提示詞

## 🎯 立即體驗功能

### 測試 1：使用預設提示詞

1. 在 ChatGPT 頁面點擊擴充功能圖示
2. 選擇「SEO 文章撰寫」
3. 填寫變數：
   - 主題：人工智慧
   - 關鍵字：AI 應用
   - 受眾：科技愛好者
   - 字數：1000
   - 語氣風格：專業易懂
4. 查看自動插入的完整提示詞

### 測試 2：建立自己的提示詞

1. 點擊「+ 新增提示詞」
2. 填寫：
   - 名稱：`快速摘要`
   - 分類：`工具`
   - 內容：`請用 [字數] 字總結以下內容：[內容]`
3. 儲存後立即使用

### 測試 3：匯出備份

1. 點擊右上角下載圖示
2. 儲存 JSON 檔案
3. 打開檔案查看內容

## 🎨 自訂外觀（選用）

### 修改主題色

編輯 `popup.css`，找到 `:root` 區塊：

```css
:root {
  --primary-color: #10a37f;  /* 改成您喜歡的顏色 */
}
```

### 修改浮動按鈕位置

編輯 `content.css`，找到 `.prompt-quick-btn`：

```css
.prompt-quick-btn {
  bottom: 24px;  /* 距離底部 */
  right: 24px;   /* 距離右側 */
}
```

## 🔧 快速除錯

### 問題：擴充功能無法載入

**檢查：**
```
1. 是否所有檔案都在資料夾中？
2. manifest.json 格式是否正確？
3. 圖示檔案是否存在？
```

**解決：**
重新下載所有檔案，確認沒有遺漏

### 問題：無法在 ChatGPT 插入

**檢查：**
```
1. 是否在對話頁面（不是首頁）？
2. 頁面是否完全載入？
3. 是否有其他擴充功能衝突？
```

**解決：**
1. 重新整理 ChatGPT 頁面
2. 前往 `chrome://extensions/` 重新載入擴充功能

### 問題：提示詞消失

**檢查：**
```
1. 是否誤刪除？
2. 是否清除了瀏覽器資料？
```

**解決：**
定期使用匯出功能備份

## 💡 使用技巧

### 技巧 1：善用變數

不要寫死內容，使用變數讓提示詞更靈活：

❌ 不好的做法：
```
請幫我寫一篇關於 Python 的文章
```

✅ 好的做法：
```
請幫我寫一篇關於 [程式語言] 的文章，
適合 [程度] 的讀者，字數約 [字數] 字
```

### 技巧 2：建立提示詞系列

為不同任務建立系列提示詞：

- `寫作-部落格-開頭`
- `寫作-部落格-段落`
- `寫作-部落格-結尾`

### 技巧 3：使用分類管理

使用一致的分類命名：

- `寫作`
- `程式設計`
- `行銷`
- `學習`
- `商業`

### 技巧 4：定期備份

建議每週或每月匯出一次，檔案很小不佔空間。

### 技巧 5：分享給團隊

匯出 JSON 檔案可以分享給團隊成員使用。
