# 安裝與設定指南

## 📋 安裝前準備

### 1. 準備圖示檔案

您需要準備三個不同尺寸的圖示檔案，放在 `icons` 資料夾中：

- **icon16.png** - 16x16 像素（用於擴充功能列表）
- **icon48.png** - 48x48 像素（用於擴充功能管理頁面）
- **icon128.png** - 128x128 像素（用於 Chrome 線上應用程式商店）

#### 建立圖示的方式：

**方式 1：使用線上工具**
1. 前往 [Canva](https://www.canva.com) 或 [Figma](https://www.figma.com)
2. 建立 128x128 像素的設計
3. 設計一個簡單的聊天氣泡圖示
4. 匯出為 PNG 格式
5. 使用圖片編輯器調整為三個尺寸

**方式 2：使用 Favicon 產生器**
1. 前往 [Favicon.io](https://favicon.io/)
2. 選擇「Text」或「Image」
3. 設計您的圖示
4. 下載並重新命名檔案

**方式 3：使用簡單的替代方案**
如果暫時沒有圖示，可以使用純色方塊：
- 建立三個純色 PNG 檔案
- 背景色：#10a37f（ChatGPT 綠色）
- 儲存為對應的檔名和尺寸

### 2. 檔案檢查清單

確認以下檔案都存在：

```
✅ manifest.json
✅ popup.html
✅ popup.css
✅ popup.js
✅ content.js
✅ content.css
✅ background.js
✅ storage.js
✅ icons/icon16.png
✅ icons/icon48.png
✅ icons/icon128.png
✅ README.md
```

## 🚀 安裝步驟

### 步驟 1：開啟擴充功能管理頁面

1. 開啟 Google Chrome 瀏覽器
2. 在網址列輸入：`chrome://extensions/`
3. 按 Enter 鍵

### 步驟 2：啟用開發人員模式

1. 在擴充功能頁面右上角
2. 找到「開發人員模式」開關
3. 將其切換為**開啟**狀態

### 步驟 3：載入擴充功能

1. 點擊「載入未封裝項目」按鈕（Load unpacked）
2. 在檔案選擇視窗中，選擇專案資料夾：
   ```
   D:\G股網\提示詞外掛-chrome extention-prompt plugin
   ```
3. 點擊「選擇資料夾」

### 步驟 4：驗證安裝

安裝成功後，您應該會看到：

✅ 擴充功能出現在列表中，名稱為「ChatGPT 提示詞管理工具」
✅ 瀏覽器工具列出現擴充功能圖示
✅ 沒有任何錯誤訊息

## ✅ 測試擴充功能

### 測試 1：開啟管理介面

1. 點擊瀏覽器工具列的擴充功能圖示
2. 應該會彈出提示詞管理視窗
3. 應該看到 3 個預設的範例提示詞

### 測試 2：新增提示詞

1. 點擊「+ 新增提示詞」
2. 填寫以下資訊：
   - 名稱：測試提示詞
   - 分類：測試
   - 內容：這是一個測試 [變數]
3. 點擊「儲存」
4. 確認提示詞出現在列表中

### 測試 3：在 ChatGPT 使用

1. 前往 [https://chat.openai.com](https://chat.openai.com)
2. 確認右下角出現綠色浮動按鈕
3. 點擊擴充功能圖示
4. 選擇一個提示詞
5. 如果有變數，填寫變數值
6. 確認提示詞被插入到輸入框

## 🔧 常見安裝問題

### 問題 1：「找不到圖示」錯誤

**症狀：**
```
Could not load icon 'icons/icon16.png'
```

**解決方式：**
1. 確認 `icons` 資料夾存在
2. 確認圖示檔案存在且名稱正確
3. 暫時建立空白圖示檔案（任何 PNG 圖片即可）

### 問題 2：擴充功能無法載入

**症狀：**
紅色錯誤訊息，顯示 manifest 錯誤

**解決方式：**
1. 檢查 `manifest.json` 是否有語法錯誤
2. 確認所有引用的檔案都存在
3. 檢查檔案編碼是否為 UTF-8

### 問題 3：無法在 ChatGPT 插入文字

**症狀：**
點擊提示詞後沒有反應

**解決方式：**
1. 確認您在 ChatGPT 對話頁面（不是首頁）
2. 重新整理 ChatGPT 頁面
3. 檢查瀏覽器 Console 是否有錯誤訊息
4. 重新安裝擴充功能

### 問題 4：找不到浮動按鈕

**症狀：**
在 ChatGPT 頁面看不到右下角的綠色按鈕

**解決方式：**
1. 重新整理頁面
2. 檢查 `content.css` 是否正確載入
3. 檢查是否被其他擴充功能遮擋

## 🔄 更新擴充功能

當您修改程式碼後：

1. 前往 `chrome://extensions/`
2. 找到「ChatGPT 提示詞管理工具」
3. 點擊重新整理圖示 🔄
4. 重新整理正在使用的 ChatGPT 頁面

## 🗑️ 解除安裝

1. 前往 `chrome://extensions/`
2. 找到「ChatGPT 提示詞管理工具」
3. 點擊「移除」按鈕
4. 確認移除

**注意：** 解除安裝會刪除所有儲存的提示詞，建議先匯出備份！

## 📦 打包為 .crx 檔案（選用）

如果想要分享給其他人使用：

1. 前往 `chrome://extensions/`
2. 確認開發人員模式已開啟
3. 點擊「封裝擴充功能」
4. 選擇專案資料夾
5. 點擊「封裝擴充功能」
6. 會產生 `.crx` 和 `.pem` 檔案
